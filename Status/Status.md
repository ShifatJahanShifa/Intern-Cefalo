# Status Code

### 2XX
`200`- **OK**  
`201`- **Created**   
`202`- **Accepted**

### 3XX
`303`- **See Other**

### 4XX
`404`- **Not Found**   
`405`- **Method Not Allowed**   
`406`- **Not Acceptable**  
`410`- **Gone**

### 5XX
`500`- **Internal Server Error** [also includes error from DataBase]
`503`- **Service Unavailable**
